# Online Banking Project - User Stories

**Project:** Online Banking System  
**Domain:** Banking Services  
**Total User Stories:** 21  
**Total Acceptance Criteria:** 64  

---

## BNK-001: Secure Login

**User Story:** As a customer, I want to log in securely so that I can access my bank account.

**Acceptance Criteria:**
1. Login requires username and password
2. Two-factor authentication is available
3. Invalid credentials show error
4. Password is encrypted
5. Session timeout after inactivity

---

## BNK-002: View Account Balances

**User Story:** As a customer, I want to view account balances so that I know my financial status.

**Acceptance Criteria:**
1. All accounts show current balances
2. Balances update in real-time
3. Account summary is displayed on dashboard
4. Pending transactions are shown
5. Historical balance is available
